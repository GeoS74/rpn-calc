# RPN Calc

Консольный калькулятор, использующий обратную польскую запись (RPN).
Реализован на языке Си в рамках решения задач 4.3-4.10 из книги K&R.

### Возможности

- Поддержка базовых операций: +, -, *, /, % (остаток от деления)
- Математические функции: sin, cos, exp, pow
- Переменные:  
    - ans — последний результат вычислений
    - _ — последнее значение, выведенное командой print
- Команды для работы со стеком:  
    - print — печать верхнего элемента без извлечения
    - duplicate — дублирование вершины стека
    - swap — обмен двух верхних элементов
    - clear — очистка стека
- Поддержка отрицательных чисел (через 0 n -)
- Работа с числами с плавающей точкой
- Обработка ошибок (деление на ноль, переполнение стека и т.д.)

### Сборка

```bash
make
```

Исполняемый файл rpncalc появится в корневой папке.

### Запуск

```bash
./rpncalc
```

Ввод построчный. Каждая строка должна содержать выражение в обратной польской записи.

Пример

```text
3 4 +
ans 2 *
3 4 + print 6 5 - * _ 2 + +
_ 5 +
ans 3 +
5.5 2.5 %
```
